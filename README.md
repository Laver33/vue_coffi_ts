# Vue 3 + TypeScript 
🎯 О проекте

Данный проект реализован для портфолио Vue3 + TS разработчика, на момент января 2025 находится в готовности на 90%

🏗️ Архитектура

- Vue 3 Composition API 
- TypeScript 
- Vue Router
- Компонентный подход 
- Адаптивный дизайн

Структура проекта:

src/
├── components/        # Переиспользуемые компоненты
│   ├── Header.vue     # Навигация с корзиной
│   ├── Footer.vue     # Футер сайта
│   ├── BestCoffi.vue  # Каталог кофе
│   ├── Different.vue  # Преимущества
│   ├── Discover.vue   # Главный баннер
│   └── Hero.vue       # Герой-секция
├── pages/             # Страницы приложения
│   ├── Home.vue       # Главная страница
│   ├── Menu.vue       # Меню кофейни
│   ├── Profile.vue    # Личный кабинет
│   ├── Orders.vue     # История заказов
│   ├── Settings.vue   # Настройки
│   ├── About.vue      # О нас
│   └── Contact.vue    # Контакты
├── router/            # Роутер
├── assets/            # Изображения и стили
└── composables/       # Функции
